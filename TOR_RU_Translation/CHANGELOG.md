# TOR — Русский перевод — Список изменений

## v1.3.15.7 — 2026-04-20

Крупная чистка накопленного транслит-мусора, ошибок перевода и
структурных багов. Порядка 300+ правок через все ~16 000 строк перевода.

### Критичные правки (видны сразу в игре)

- **Тиры рыцарства Бретоннии** в тултипе боевого духа отряда больше не
  транслит, а настоящие слова:
  - `Уннигтли` → **«Бесчестный»**
  - `Хоноурабле` → **«Благородный»**
  - `Нотеворти` → **«Заслуженный»**
  - `ПуреХеартед` → **«Чистосердечный»**
  - `Унинспиринг` → **«Посредственный»**
  - `пинкере` → **«Искренний»**
  - `Рыцарство Ранк` → **«Ранг рыцарства»**
- **Уровень Ваагх** больше не `Унновн` — теперь **«Неизвестно»**.
- **Диалоги чарователей** всех фракций (Бретоннии, Эонира, Асрай, Империи):
  14 строк `TODO-плейсхолдеров` заменены на осмысленные реплики.

### Ink-квесты больше не сыпят английским

Раньше текст `Traveling Merchant`, `Cabin in the Woods Locked`, `Фэйр`,
`Поле боя`, `Минстрел`, `Duel`, `Meadow` и др. показывался по-английски,
если игрок не запустил `УСТАНОВИТЬ_ink.bat`. Теперь:

- **123 новых XML-строки** для `#STR_`-тегов всех 27 ink-квестов — перевод
  подхватывается автоматически без ручной установки ink-файлов.
- `УСТАНОВИТЬ_ink.bat` полностью переписан — раньше мог закрываться
  мгновенно из-за проблем с кодировкой / line endings / скобок в пути
  `(x86)`. Теперь ASCII, CRLF, безопасные `call :try` вместо `for`,
  `delayed expansion` `!VAR!` вместо `%VAR%` чтобы `&` в «Mount & Blade»
  не ломал парсер.
- Запуск bat нужен только для 12 квестов без тегов («Да Босс
  Пробуждается», орочьи и нурглистские видения и т.п.) — для остальных
  RU подтягивается из XML.

### Имперские дезертиры — инверсия смысла

В оригинале `Deserter / Deserter Sergeant / Deserter Scout` (бандиты-
дезертиры), в переводе было `Добровольный Ополченец` — противоположный
смысл. Исправлено:

- → **«Дезертир»**, **«Сержант-дезертир»**, **«Разведчик-дезертир»**
- `Империя Десертер Клан` → **«Клан Имперских Дезертиров»**
- `Имперская Государственная Войска` → **«Имперский государственный солдат»**

### Голый английский в русских строках

- `Cairn → Каирн` (доспехи призрака ×2)
- `Flagellant → Флагеллянт` (крафт-компоненты ×5)
- `Voulgier → вульж` (бретонская алебарда)
- `Heater → геральдический` (5 вариантов щитов + «окованный»)
- `bastard-меч → полуторный меч`
- `fuller → дол` (клинок с узким долом)
- `intruder → чужак` (описание поселения)
- `helpnecromancersuccess/fail` — раньше в тексте был id предмета
  (`tor_vc_weapon_staff_nm_001`), теперь нормальные фразы.

### Транслит-мусор в названиях и описаниях

- `Лигтнинг → Молния` (×4) — урон заклинаний, описания карьер, печать Ульрика
- `Флюэнтно → Бегло` (предыстория Империи)
- `Фигт! → В бой!` (пещера тролля)
- `Унстоппабле → Неудержимый` (печать Ульрика)
- `Рангед → Дальний бой`, `Магикал → Магический`, `Фисикал → Физический`
- `Пройектиль → Снаряд`, `Праиинг → Молитва`, `Енаблед → Включено`
- `Лоот → Добыча`, `Аддитионал → Дополнительный`
- `ундерстанд/Ундерстанд → понимаю/Понял`
- `Сквигт → Сквиг`, `Енд → Конец`

### Подмена заглавных букв (С→п, З→а) в капс-тексте

Ранее из-за визуально похожих букв в ряде КАПС-фраз заглавные «С» и «З»
были заменены строчными. Восстановлены:

- `МЯпО → МЯСО` (тролль)
- `пЮДА → СЮДА`, `аЕЛЁНЫЕ → ЗЕЛЁНЫЕ` (прощание зеленокожего)
- `ПОЖАЛУЙпТА → ПОЖАЛУЙСТА`, `МЕпТЕ → МЕСТЕ` (жрец Ульрика)
- `МАГИЧЕпКИЕ пПОпОБНОпТИ → МАГИЧЕСКИЕ СПОСОБНОСТИ` (Кровавый Поцелуй)
- `анаюсь → Владею`, `анаете → Знаете`
- `ПОДпТУПОВ → ПОДЛЮГ`, `пюда → сюда`, `ДИп → ДИС`

### Перки карьер — HP-единица и термины магии

~40 перков разных карьер (рыцари, маги, орки-боссы, вампиры, жрецы)
использовали «+X жизней» / «жизненных очков» / «жизненным силам» — неверный
перевод английского `health`. Везде теперь корректно **«здоровья»**:

- `+10 жизней` → `+10 здоровья` (Дуэлянт, Выживший и др.)
- `+15 личных жизней` → `+15 личного здоровья`
- `восстанавливает личные жизни` → `полностью восстанавливает здоровье`
- `+50% жизней личного коня` → `+50% здоровья личного коня`
- `восстанавливает всем +50 жизней` → `+50 здоровья всем`

Плюс перки Имперских школ магии:

- `Арканного канала / Арканного Кондуита` → **«канала Тайной магии»**
- `Арканная канализация` → **«Поток тайной магии»**
- `Арканные/Арканские свитки` → **«магические свитки»**
- Имя тома `Де Арканис Кадон` оставлено как имя собственное.

И прочие:

- `Однажды в день отправляйтесь на охоту.` → **«Раз в день отправляйтесь на охоту.»**
- **Китайские иероглифы `单手`** в RU-строке → **«одноручного»**
  (перк «Хранитель Торговани»)
- `войск «Меньшие нежитьи»` → **«Меньшая нежить»** / `«Меньшей нежити»` (8 строк)
- `+1 жизнь` → `+1 здоровья`, опечатка `аа каждый` → `За каждый`

### Дубликаты строк поселений (RoR-sidecar)

Обнаружены и ликвидированы **88 дублирующихся ID** между файлами
`str_tor_ror_settlement_templates-rus.xml` и `str_tor_settlements-rus.xml`
с расходящимся текстом (двигло рандомно выбирало вариант). Для каждого
выбран лучший перевод, попутно исправлены канон-ошибки:

- `Имперский Гуннери Шоол → Имперская школа артиллерии`
- `Остланд Блак Стражник → Чёрная стража Остланда`
- `Конигштейн Крипт → Крипта Кёнигштейна`
- `Патруль Ривер → Патруль реки Стир`
- `Форготтен Легион Крипт → Крипта Легиона Забытых`
- `Лонгшанк Енкампмент → Лагерь Лонгшанкс`
- `вардансеров → Танцоров Войны` (канон Wardancers)
- `Смертных Джек → Смертоджеков` (канон Deathjacks)
- … и 80 ещё

Дополнительно: 5 одинаковых дублей вычищены, `Орка-Селянка` унифицирована.

### Грамматика род/число

- `{HERO_NAME} стал Вампиром-Крестоносцем → становится Рыцарем-Вампиром Крови`
- `{HERO_NAME} стал Нексархом → становится Некрархом`
- Диалоги игрока с инженером Нульна и гномом-скитальцем — перефразированы
  без маскулинных глаголов прошедшего времени (работает и для женщин).

### Прочие точечные правки

- `+40% лигтнинг Урон → +40% к урону молнии`
- `+20% Лигтнинг Урон → +20% к урону молнии` (печать Мананна)
- `СвежаяКровь → Свежая Кровь` (хузаитский лорд)
- `«плотлингов» → «снотлингов»` (предысторика зеленокожего)

### Итого

~300 исправлений через все 34 XML-файла (16 028 строк) + фикс установочного
bat-скрипта. Полная проверка XML-целостности — 0 структурных ошибок.

### Что осталось намеренно нетронутым (канон/стилистика)

- `Унгрим / Унгрима` — каноничное имя Ungrim Ironfist.
- `Бриархеарт`, `Курноур` — имена Лесных Эльфов Asrai, намеренный flavor.
- `ЛеРои` — французский стиль имени герцогини.
- `WAAAGH` в капсе лорных описаний — WH-стилистика.
- Орочий и вампирский lexicon (варбосс, снотлинг, халфлинг, ваагх,
  варпстоун) — канон WH, не русифицируется.
