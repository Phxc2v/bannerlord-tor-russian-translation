### Восстановление bulk-sed артефактов (массовая потеря букв)

При прошлых заменах sed-скриптами были случайно «пожраны» начальные символы в десятках строк. Скриптом-валидатором `find_translation_bugs.ps1` найдены и точечно восстановлены:

- **Артефакт «С→п»** (заглавная С в начале слов превратилась в «п»): `пегодня→Сегодня`, `пейчас→Сейчас`, `пердце→Сердце`, `пуществует→Существует`, `пуществуют→Существуют`, `пам→Сам`, `пекреты→Секреты`, `поберите→Соберите`, `побирайте→Собирайте`, `побрат-маг→Собрат-маг`, `повременник→Современник`, `полнечный→Солнечный`, `поревнование→Соревнование`, `постояние→Состояние`, `пудя→Судя`, `пъешь→Съешь`, плюс 14× одиночное `п ` в начале строк («п честью клянусь», «п тяжёлым сердцем», «п самого начала», «п тех пор как», «п нетерпением», «п непоколебимой», «п дарами лорда Чумы», «п помощью древних томов», «п Топориками», «п бородой», «п завершением» и пр.) → восстановлено до «С …».
- **Артефакт «З→а» / «За→аа»**: `анак→Знак`, `анака→знака`, `анай→Знай` (×3), `анайте→Знайте`, `аначит→Значит`, `анание→Знание`, и **15+ строк описаний абилок** «аа каждый/каждые/каждое уровень…» → «За каждый/каждые/каждое уровень…».
- **Потеря «Ж»**: `наковальням ахуфбара` → `Жуфбара` (str_tor_feudal_oath_line2_zhufbar).
- **Потеря «д»**: `в Гилью` (×2) → `в Гильдию`.
- **Потеря «о»**: `мой ремесл` → `моё ремесло`.

## v1.3.15.2 (2026-04-18)

**Архитектура — перевод стал отдельным модулем**

Раньше перевод распространялся как sidecar-XML, который надо было кидать поверх файлов TOR_Core (затирался при апдейтах). Теперь это самостоятельный модуль: ставится рядом, включается галочкой, не ломается при обновлениях TOR. Переводы Core и Armory объединены в одном модуле.

**Переведены захардкоженные UI-элементы**

Часть текстовых меток в модах Bannerlord не идёт через словарь локализации — они прошиты прямо в префабах интерфейса. Эти места раньше оставались английскими даже при включённом русском. В этом релизе 4 префаба переопределены:

- **Инвентарь (тултипы):** `Item Effects` → Эффекты предмета, `Magic Item` → Магический предмет
- **Зачарование:** `Items / Enchantments / Enchant / Done` → Предметы / Чары / Зачаровать / Готово
- **Книга заклинаний:** `SpellBook / Spellcasting Stats / Done` → Книга заклинаний / Характеристики заклинателя / Готово
- **Книга боевых молитв:** `Battle Prayers / Priest Stats / Done` → Боевые молитвы / Характеристики жреца / Готово

**Массовые правки повреждённых слов**

- `плушайте / плужба / плужить` → Слушайте / Служба / Служить
